Tutorial

1. Connect your micropython board which had connect with LoRa module to your computer
2. Open Thonny (Python IDE) (If you don't have Thonny you can download from here)
3. Click "Options..." in "Tools" menu
4. เลือกบอร์ดและพอร์ตในการเชื่อมต่อที่ใช้ (ในที่นี้คือ Micropython(ESP32) เชื่อมต่อที่ Serial Port @COM4)
5. Press (Ctrl+N) to create a new blank file
6. Then use this code to get DeviceEUI from LoRa module 
                    from machine import UART
                    import time
                    # UART2
                    # IO Pin:Tx = 16
                    # IO Pin:Rx = 17
                    uart = UART(2, baudrate=115200, bits=8, parity=None, stop=1, timeout=2000)
                    uart.write("AT+DEVEUI\r\n")
                    time.sleep(1)
                    print(uart.read().decode('utf-8'))
